                    ШАГ 1: ПОЛУЧЕНИЕ ТОКЕНОВ
▰▰▰▰▰▰▰▰▰▰▰▰▰▰▰▰▰▰▰▰▰▰▰▰▰▰▰▰▰▰▰▰▰▰▰▰▰▰▰▰▰▰▰▰▰▰▰▰▰▰▰▰▰▰▰▰▰▰▰▰

1. ТОКЕН БОТА (BOT_TOKEN):
   • Откройте Telegram
   • Найдите бота @BotFather
   • Напишите /newbot
   • Придумайте имя боту (например: Мой Магазин)
   • Придумайте юзернейм (например: myshop_bot)
   • Скопируйте токен вида: 7123456789:AAHxxx...xxx

2. ТОКЕН CRYPTOBOT (CRYPTOBOT_TOKEN):
   • Найдите бота @CryptoBot в Telegram
   • Нажмите "Crypto Pay"
   • Нажмите "Create App"
   • Придумайте название приложения
   • Скопируйте API Token

3. ВАШ ID (ADMIN_IDS):
   • Найдите бота @userinfobot
   • Напишите ему /start
   • Скопируйте ваш ID (только цифры)

                  ШАГ 2: УСТАНОВКА PYTHON
▰▰▰▰▰▰▰▰▰▰▰▰▰▰▰▰▰▰▰▰▰▰▰▰▰▰▰▰▰▰▰▰▰▰▰▰▰▰▰▰▰▰▰▰▰▰▰▰▰▰▰▰▰▰▰▰▰▰▰▰

▶ WINDOWS:
  1. Скачайте Python: https://www.python.org/downloads/
  2. Запустите установщик
  3. ⚠️ ВАЖНО: Поставьте галочку "Add Python to PATH"
  4. Нажмите "Install Now"

▶ MAC:
  1. Откройте Терминал (Finder → Программы → Утилиты → Терминал)
  2. Введите команду:
     brew install python3
  
  Если brew не установлен, сначала выполните:
     /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

▶ UBUNTU / LINUX:
  Откройте Терминал и введите:
     sudo apt update
     sudo apt install python3 python3-pip

                   ШАГ 3: ПОДГОТОВКА ФАЙЛОВ
▰▰▰▰▰▰▰▰▰▰▰▰▰▰▰▰▰▰▰▰▰▰▰▰▰▰▰▰▰▰▰▰▰▰▰▰▰▰▰▰▰▰▰▰▰▰▰▰▰▰▰▰▰▰▰▰▰▰▰▰

1. Создайте папку для бота (например: C:\bot или ~/bot)

2. Скопируйте в эту папку файлы:
   • digital_shop_bot.py
   • requirements.txt

3. Создайте файл .env (именно так, с точкой в начале!)
   
   ⚠️ WINDOWS: 
   Создайте файл через Блокнот, при сохранении выберите 
   "Все файлы (*.*)" и назовите файл: .env

4. Откройте .env и вставьте (замените на свои данные):

BOT_TOKEN=7123456789:AAHxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx
CRYPTOBOT_TOKEN=12345:AAxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx
ADMIN_IDS=123456789
SUPPORT_USERNAME=@ваш_юзернейм
SHOP_NAME=Название Магазина

   ⚠️ Без пробелов вокруг знака =
   ⚠️ Без кавычек

КОМАНДЫ:
  /start — Главное меню (для всех)
  /admin — Админ-панель (только для админов)

КНОПКИ В МЕНЮ:
  🛒 Купить        — Каталог товаров
  👤 Мой профиль   — Профиль и покупки
  🏬 О шопе        — Информация о магазине
  🛟 Поддержка     — Связь с поддержкой

АДМИН-ПАНЕЛЬ:
  📊 Статистика    — Пользователи, продажи, выручка
  🖼 Медиа         — Картинки/видео для разделов
  📨 Рассылка      — Отправка сообщений всем
  📦 Товары        — Управление товарами
  📁 Категории     — Управление категориями
  ⚙️ Настройки     — Описание магазина

ДОБАВЛЕНИЕ ТОВАРА:
  1. /admin → Категории → Добавить категорию
  2. /admin → Товары → Добавить товар
  3. Выберите категорию
  4. Введите название, описание, цену
  5. Выберите тип (текст или файл)
  6. Отправьте контент товара

                   ЧАСТЫЕ ОШИБКИ И РЕШЕНИЯ
▰▰▰▰▰▰▰▰▰▰▰▰▰▰▰▰▰▰▰▰▰▰▰▰▰▰▰▰▰▰▰▰▰▰▰▰▰▰▰▰▰▰▰▰▰▰▰▰▰▰▰▰▰▰▰▰▰▰▰▰

❌ "python не является внутренней командой"
   → Python не добавлен в PATH
   → Переустановите Python с галочкой "Add to PATH"

❌ "No module named aiogram"
   → Не установлены библиотеки
   → Выполните: pip install -r requirements.txt

❌ "FileNotFoundError: .env"
   → Файл .env не создан или не в той папке
   → Создайте .env в папке с ботом

❌ "Unauthorized"
   → Неверный BOT_TOKEN
   → Проверьте токен в .env

❌ "ADMIN_IDS error"
   → Неверный формат ID
   → Укажите только цифры без пробелов

❌ Бот не отвечает
   → Проверьте, что бот запущен в терминале
   → Проверьте, что написали боту /start
